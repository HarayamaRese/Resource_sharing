周巨展，男，浙江瑞安陶山人。1980年至今已发表各类文字80余万字，作品获奖20余次。正式出版的著作有文集《陶山风景》、长篇纪实小说《灵魂冲击波》、与人合作主编《瑞安民间故事》等。现为《中国经济》杂志驻浙南（温台丽）记者。
